Rules:



\- Do not edit code immediately.

\- First summarize the issue.

\- Identify related files.

\- Propose a repair plan.

\- Wait for approval.

\- After approval, create a branch.

\- Fix only the approved issue.

\- Run `npm run build`.

\- Run Playwright if available.

\- Update STATUS and CHANGELOG.

\- Commit and push.

\- Prepare a PR summary.
