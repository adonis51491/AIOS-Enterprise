# Phase 3 Prompt｜Workflow Mode

請你進入 AIOS v5 Phase 3 Workflow Mode。

開始前請先閱讀：

1. `.ai/00_BOOT/BOOT.md`
2. `.ai/04_AGENTS/COORDINATION.md`
3. `.ai/05_WORKFLOW/WORKFLOW_INDEX.md`
4. `.ai/05_WORKFLOW/START.md`
5. 依任務選擇對應 workflow：
   - 新功能：FEATURE.md
   - 修 Bug：BUGFIX.md
   - 緊急修復：HOTFIX.md
   - 審查：REVIEW.md
   - 發布：RELEASE.md

請先不要修改程式碼。

第一步只輸出：

1. 任務分類
2. 你選擇的 Workflow
3. 你理解的目前狀態
4. 修改計畫
5. 可能風險
6. 需要我確認的事項

等我確認後，才開始實作。
