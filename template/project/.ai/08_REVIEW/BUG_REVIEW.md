\# Bug Review Gate



Before editing code, AI must answer:



1\. What is the selected Bug ID?

2\. What module does it belong to?

3\. What files will be modified?

4\. What database or schema risks exist?

5\. What race condition or consistency risks exist?

6\. What backward compatibility risks exist?

7\. What security risks exist?

8\. What acceptance criteria will prove the fix works?

9\. What rollback method is available?



AI must not edit code until this review gate is answered and approved.
