# Playwright Automation

Playwright Agent 負責：

1. 啟動或確認 localhost。
2. 開啟首頁。
3. 檢查 Desktop / Tablet / Mobile。
4. 讀取 Console Error。
5. 讀取 Network Error。
6. 截圖。
7. 產生 QA Report。

標準尺寸：

- Desktop: 1440 x 900
- Tablet: 768 x 1024
- Mobile: 375 x 812

檢查目標：

- Header / Navbar
- Hero
- CTA
- Main Content
- Footer
- Console
- Layout Shift
