# Retrospective Workflow｜回顧流程

## 使用時機

- 每個 Sprint 結束
- 重大 Bug 修復後
- Release 後
- AI 連續修壞時

## 回顧問題

1. 這次做對了什麼？
2. 哪裡卡住？
3. 哪些 Prompt 不清楚？
4. 哪些文件需要補？
5. 哪些規則應該加入 MEMORY.md？
6. 哪些測試應該自動化？

## 輸出格式

```markdown
# Retrospective

日期：
範圍：

## 做得好
- 

## 問題
- 

## 根本原因
- 

## 改進行動
- 

## 要加入 MEMORY.md 的規則
- 
```
