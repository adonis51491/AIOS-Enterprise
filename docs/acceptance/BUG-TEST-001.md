# Acceptance Criteria: BUG-TEST-001

## Must Pass

- The observed issue is addressed.
- The expected behavior is satisfied.
- The fix is limited to the affected scope.
- No unrelated files are modified.
- Verification result is documented.

## Verification Steps

1. Reproduce or clearly explain the issue.
2. Apply the repair.
3. Verify the expected behavior.
4. Confirm no regression in nearby behavior.
